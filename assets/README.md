# Assets Folder

This folder should contain:

1. **icon.png** - App icon (1024x1024)
2. **splash.png** - Splash screen (1284x2778)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024)
4. **favicon.png** - Web favicon (48x48)
5. **player-placeholder.png** - Fallback for player images (200x200)

## Quick Setup

You can use any placeholder images for development. For production:

1. Create a basketball-themed icon
2. Use dark background (#0A0A0A) for splash
3. Add your app name/logo
